Roombeat Static Site

Files:
- index.html
- favicon.png

How to run locally:
- In this folder, run: python3 -m http.server 3000
- Then open: http://localhost:3000 in your browser.

How to deploy to Vercel:
- Create a new project in Vercel and link this folder via Git (GitHub/GitLab/Bitbucket),
  or upload the folder contents to a new static project.
